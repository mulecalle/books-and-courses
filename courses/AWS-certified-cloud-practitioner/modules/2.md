# Computación en la nube

La computación en la nube significa crear máquinas virtuales con un proveedor de la nube para ejecutar aplicaciones y tareas a través de internet

## Amazon EC2

Types:
- De uso general
- Optimizadas para la computación
- Optimizadas para memoria
- De computación acelerada
- Optimizadas para el almacenamiento

Los hosts dedicados ofrecen el uso exclusivo de un servidor con control total, mientras que las instancias dedicadas proporcionan aislamiento sin control sobre el servidor.

La escalabilidad se refiere al potencial de un sistema para crecer con el tiempo, mientras que la elasticidad se refiere al ajuste dinámico y bajo demanda de los recursos.

## Elastic Load Balancing (ELB)

Elastic Load Balancing (ELB) distribuye automáticamente el tráfico de aplicaciones entrantes en varios recursos, como las instancias de EC2, para optimizar el rendimiento y la fiabilidad. Un equilibrador de carga actúa como un único punto de contacto para todo el tráfico web que entra en un grupo de escalado automático.

Types:
- Round Robin, Distribuye el tráfico de manera uniforme entre todos los servidores disponibles de forma cíclica.
- Least Connections: Dirige el tráfico al servidor con el menor número de conexiones activas, lo que mantiene una carga equilibrada.
- IP Hash: Usa la dirección IP del cliente para dirigir el tráfico de manera uniforme al mismo servidor.
- Least Response Time: Dirige el tráfico al servidor con el tiempo de respuesta más rápido, lo que minimiza la latencia.

## EventBridge

EventBridge es un servicio sin servidor que ayuda a conectar diferentes partes de una aplicación mediante eventos, lo que permite crear sistemas escalables y basados en eventos

## Amazon Simple Queue Service (Amazon SQS)

Amazon SQS es un servicio de colas de mensajes que facilita una comunicación fiable entre los componentes de software. Puede enviar, almacenar y recibir mensajes a cualquier escala, garantizando que no se pierdan y sin depender de la disponibilidad de otros servicios para procesarlos. En Amazon SQS, una aplicación coloca los mensajes en una cola, y un usuario o servicio recupera el mensaje, lo procesa y, a continuación, lo elimina de la cola.

## Amazon Simple Notification Service (Amazon SNS)

Amazon SNS es un servicio de publicación y suscripción que los publicadores utilizan para enviar mensajes a los suscriptores a través de temas de SNS. En Amazon SNS, los suscriptores pueden incluir servidores web, direcciones de correo electrónico, funciones de Lambda y otros puntos de conexión.
