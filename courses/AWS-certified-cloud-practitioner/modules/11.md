# Precios y soporte

## Precios AWS

### Pago por uso

Con el pago por uso, puede adaptarse a las cambiantes necesidades empresariales y reducir el riesgo de aprovisionar en exceso o sufrir una falta de capacidad.

### Ahorro cuando se compromete

Para ciertos servicios, como los servicios de computación de AWS, Savings Plans ofrece descuentos en comparación con los precios bajo demanda si se compromete a un plan de 1 o 3 años.

### Más uso, menos dinero

Con AWS, puede obtener importantes ahorros a medida que aumenta su uso. Para algunos servicios, los precios son escalonados, lo que significa que cuanto más los utilice, menos pagará.

Factores determinantes del costo

## El precio de los servicios de AWS varía en función de varios factores, como la categoría o el tipo de servicio, la configuración, la región de AWS y el modelo de precios que elija.
Existen tres aspectos fundamentales de costo con AWS: el cómputo, el almacenamiento y la transferencia de datos salientes. Estos factores determinantes afectan las categorías de servicios de AWS de diferentes maneras.

### Precios y facturación de AWS

#### AWS Organizations
AWS Organizations proporciona administración y gobernanza centralizados de su entorno de AWS. Con AWS Organizations, puede crear, agrupar y administrar cuentas. También puede aplicar políticas de seguridad a nivel de cuenta y consolidar la facturación con varias cuentas mediante un único método de pago.

#### Panel de administración de facturación y costos de AWS
El Panel de administración de facturación y costos de AWS centraliza la administración de costos y muestra los cargos actuales, el uso, las previsiones y los desgloses detallados. También proporciona herramientas para administrar los pagos, ver las facturas, establecer presupuestos y consolidar la facturación.

#### AWS Budgets
AWS Budgets ayuda a establecer presupuestos personalizados y envía alertas cuando los costos, el uso o la cobertura o la utilización de Savings Plans y las instancias reservadas (RI) superan los umbrales definidos.

#### Explorador de costos de AWS
El explorador de costos de AWS ayuda a visualizar, analizar y administrar los costos y el uso de AWS con gráficos, informes y pronósticos interactivos. Brinda información sobre los patrones de gasto, las tendencias y las recomendaciones de instancias reservadas.

#### Calculadora de precios de AWS
Otra herramienta útil es la calculadora de precios de AWS. La calculadora de precios de AWS es una herramienta de planificación basada en la web que puede utilizar para crear estimaciones. Puede ingresar configuraciones específicas, como tipos de instancias, opciones de almacenamiento y volúmenes de transferencia de datos. Luego, según sus configuraciones, recibirá un desglose detallado de los costos para ayudarle a presupuestar la asignación de recursos de AWS.

### AWS Marketplace

AWS Marketplace es un catálogo digital que incluye miles de listados de software de proveedores de software independientes.

### Red de socios de AWS

La Red de socios de AWS (APN) es una comunidad global que utiliza las tecnologías, los programas, la experiencia y las herramientas de AWS con el fin de crear soluciones y servicios para los clientes. Juntos, los socios y AWS brindan soluciones innovadoras, resuelven desafíos técnicos y brindan valor a los clientes.

Puede trabajar con los socios de AWS para crear o utilizar soluciones especializadas que se adapten a sus necesidades empresariales únicas.
