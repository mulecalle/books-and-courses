# soluciones de Well-Architected

## Servicios de desarrollo

### AWS CodeBuild

CodeBuild es un servicio de integración continua completamente administrado que compila código fuente, ejecuta pruebas y produce paquetes de software para su implementación.

### AWS CodePipeline

CodePipeline es un servicio de integración y entrega continuas (continuous integration, continuous delivery, CI/CD) completamente administrado que automatiza las fases de creación, prueba e implementación del proceso de lanzamiento.

### AWS X-Ray

X-Ray es una potente herramienta de rastreo, depuración y análisis del rendimiento que ayuda a los desarrolladores a visualizar el comportamiento de las aplicaciones. Con X-Ray, los desarrolladores pueden identificar rápidamente los cuellos de botella en el rendimiento, solucionar problemas y optimizar las aplicaciones para lograr una mayor eficiencia y fiabilidad.

### AWS AppSync

AWS AppSync es un servicio de GraphQL completamente administrado. Con AWS AppSync, los desarrolladores pueden crear una única API de GraphQL que pueda acceder, manipular y combinar de forma segura los datos de varios orígenes.

### AWS Amplify

Amplify es útil para optimizar el proceso de desarrollo, implementación y administración de aplicaciones de pila completa, seguras y escalables en AWS. Con Amplify, los desarrolladores pueden agregar características rápidamente, como autenticación, API, almacenamiento y alojamiento, con una administración mínima de la infraestructura.

## Servicios de aplicaciones empresariales

### Amazon Connect

Las empresas pueden usar este servicio de centro de contacto basado en IA para configurar y operar de manera eficiente un centro de llamadas de servicio al cliente escalable.

## Amazon Simple Email Service (Amazon SES)

Amazon SES es un proveedor de servicio correo electrónico escalable y rentable que puede integrarse en cualquier aplicación para automatizar el envío de correos electrónicos fiables y de alto volumen.

## Servicios de computación de usuario final

### Amazon AppStream 2.0

AppStream 2.0 es un servicio completamente administrado que transmite aplicaciones desde la nube directamente a cualquier dispositivo compatible. Esto incluye aplicaciones de software como servicio (software-as-a-service, SaaS) y aplicaciones convertidas de escritorio a SaaS sin revisiones de código.

### Amazon WorkSpaces

WorkSpaces es un servicio de computación de escritorio basado en la nube completamente administrado. Con WorkSpaces, los empleados pueden acceder de forma segura a su entorno de trabajo desde cualquier dispositivo con conexión a internet.

### Amazon WorkSpaces Secure Browser

WorkSpaces Secure Browser es un navegador empresarial remoto completamente administrado.

## Servicios de IoT

### AWS IoT Core

AWS IoT Core es un servicio de nube administrado que se utiliza para conectar dispositivos físicos con aplicaciones en la nube de forma segura.

## AWS Well-Architected

La Herramienta de AWS Well-Architected (Herramienta de AWS WA) es un servicio gratuito que ayuda a evaluar y mejorar las cargas de trabajo en la nube según los seis pilares clave: excelencia operativa, seguridad, fiabilidad, eficiencia del rendimiento, optimización de costos y sostenibilidad. Ofrece revisiones de cargas de trabajo, seguimiento de hitos y enfoques adaptados para evaluaciones personalizadas y planes de mejoras. Gracias a su integración con servicios de AWS, como AWS Identity and Access Management (AWS IAM) y las API, permite la colaboración en equipo y el seguimiento continuo del progreso.

### Excelencia operativa:
se centra en las operaciones, la supervisión, la automatización y la mejora continua.

### Seguridad:
Protege los sistemas y los datos mediante las prácticas recomendadas, como el principio de mínimo privilegio y la integridad de los datos.

### Fiabilidad:
Hace hincapié en la planificación de la recuperación y la adaptabilidad del sistema para satisfacer las demandas cambiantes.

### Eficiencia del rendimiento:
fomenta el uso de los recursos adecuados para el trabajo y los ajustes a medida que cambian las necesidades.

### Optimización de costos:
Ayuda a controlar y reducir los costos mediante el aprovisionamiento inteligente y la administración de recursos.

### Sostenibilidad:
Promueve un diseño eficiente en el uso de la energía y un uso de recursos responsable con el medio ambiente.
